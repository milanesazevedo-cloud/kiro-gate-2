# Kiro Gateway

**Language:** Python 3.10+ | **Framework:** FastAPI
**License:** AGPL-3.0

## Overview

OpenAI/Anthropic compatible API proxy for Kiro (Amazon Q Developer).

## Multi-Account Support

```bash
# Comma-separated
REFRESH_TOKEN="token1,token2,token3"

# Or numbered
REFRESH_TOKEN1="token1"
REFRESH_TOKEN2="token2"
```

## Quick Start

```bash
pip install -r requirements.txt
python main.py
```
